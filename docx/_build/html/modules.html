Модули¶

Интерфейс для получения текущей погоды через API weatherbit.

api.current_weather(city: str)¶

Возвращает текущую погоду для указанного города.

Параметры:

city (str) – Название города на любом поддерживаемом языке.

Результат:

Ответ API в виде словаря или None при ошибке/некорректном вводе.

Тип результата:

dict | None

CLI-приложение для получения и сохранения текущей погоды.

main.main()¶

Точка входа CLI.

Разбирает аргументы командной строки, получает данные погоды и выводит их в консоль или сохраняет в файлы/БД.

Утилиты для сохранения погодных данных в файлы.

save.save_json(data)¶

Сохраняет погодные данные в JSON-файл report.json.

Параметры:

data (dict) – Словарь с данными о погоде, ожидает ключи, которые возвращает API weatherbit.

save.save_txt(data)¶

Сохраняет погодные данные в текстовый файл report.txt.

Параметры:

data (dict) – Словарь с данными о погоде, ожидает ключи, которые возвращает API weatherbit.

Простые вспомогательные функции для работы с таблицей weather_data.

datadase.db_create(data)¶

Добавляет запись о погоде в таблицу weather_data.

Параметры:

data (dict) – Данные о погоде из API weatherbit.

datadase.db_delete(city)¶

Удаляет записи о городе из таблицы weather_data.

Параметры:

city (str) – Название города.

datadase.db_select(table)¶

Выводит в stdout все строки указанной таблицы.

Параметры:

table (str) – Имя таблицы базы данных.
